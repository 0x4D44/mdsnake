### Phase 1 — UNDERSTAND (done)
- Workflow `coil-understand`: 4 parallel readers (core / render / almanac / prior-art)
  → synthesizer. Brief written to `.hld-work/understand-brief.md`.
- Key constraints surfaced for a **full-game** HLD:
  - Purity is load-bearing (undo = snapshot stack; no-op via referential equality).
    Animation/transient state must stay OUT of the snapshot stack.
  - Single **rigid body**, global support — no per-segment grip hook
    (constriction/anchor/perch have nowhere to live today).
  - **Closed model surface**: `CellType` union + single-occupant cell Map; only
    `wall` supports/blocks. Carried items / overlapping entities / supportable
    objects need a model change.
  - **Single snake + global status**, head-only win → co-op not representable.
  - Core emits only final post-settle positions → smooth motion needs renderer to
    diff snapshots + persistent meshes + time loop (architectural pivot).
  - Almanac seam: `base:"./"`, committed built bundle (never hand-edit), one
    `data.js` ESSAYS entry, reuse TAGS vocab, `.nojekyll`.
- Central design problem for V1: extend the model to support all abilities
  **without premature abstraction** (the M8-ramp-to-nowhere risk).
